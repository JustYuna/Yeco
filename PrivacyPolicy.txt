# Privacy Policy — Yeco

Yeco respects user privacy.

## 1. Data Collection
The bot may store:
- User IDs
- Economy data (balance, transactions, etc.)

No personal sensitive data (such as passwords or messages) is collected.

## 2. Data Usage
Collected data is used only for:
- Economy system functionality
- Command processing
- Preventing abuse

## 3. Data Sharing
No data is sold or shared with third parties.

## 4. Data Storage
Data is stored locally using SQLite3 or equivalent database systems.

## 5. Data Removal
Users may request deletion of their stored data at any time.

This includes:
- User ID
- Economy balance and transaction history

Server data is not stored by this bot.

## 6. Changes
This policy may be updated at any time.
